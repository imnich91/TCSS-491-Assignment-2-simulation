Weather Simulation

This simulation uses a Hidden Markov Model to determine the weather for each day.

States: X = {sun, rain}

Initial Distribution: Even chance to start with sun or rain.

The transition table is as follows:

| Xt-1 | Xt | P( Xt | Xt-1 ) |

| sun | sun | .9 |

| sun | rain | .1 |

| rain | sun | .3 |

| rain | rain | .7 |

The plants need enough rain in order to survive. If the number of sunny days ever exceeds the number of rainy days by 7 days then the plants will die. They can come back to life if the number rainy days gets back within 7 days of the number of sunny days.
